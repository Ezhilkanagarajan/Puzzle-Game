# Puzzle-Game
A game which helps users to match the puzzled picture with the original one.
This project is developed using JAVA.
The user interface contains the puzzled picture as well as the original picture.The player can make moves either diagonally,left side or
right side.By making moves,the puzzled picture is made out into the original image.
After the player has finished his play and has made the original picture out of the puzzled picture,then a dialog box stating the score of the player will be displayed.
This score is based on the number of moves that the player has made.
